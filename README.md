# Student-Performance-Dashboard
📌 Project Overview

This project presents an interactive Power BI dashboard analyzing student academic performance, identifying trends, and highlighting performance distribution patterns.

📊 Key Insights

Performance distribution across marks

Average score analysis

Identification of high- and low-performing students

Attendance vs performance trends 

🛠 Tech Stack

Power BI, DAX, Data Modeling, Microsoft Excel
